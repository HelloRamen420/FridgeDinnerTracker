# 🍳 FridgeDinnerTracker - 冷蔵庫＆晩ごはん管理アプリ (iOS)

iPhoneで動作する、冷蔵庫の食材在庫管理と、作った晩ごはんのレシピ・思い出記録をシームレスに連携させた**SwiftUI製ネイティブアプリ**です。

---

## ✨ 主な機能

1.  **📊 ダッシュボード**
    *   消費期限が近い食材（2日以内）、期限が切れた食材をカラー付きカードで警告。
    *   今週の晩ごはんの作成回数、冷蔵庫のストック品数を一覧。
    *   冷蔵庫の中身から**「期限が近い食材」を自動抽出し、おすすめ料理を日替わりで提案するスマート機能**を搭載！
2.  **🥩 冷蔵庫管理**
    *   食材をカテゴリ別（肉、魚、野菜、乳製品、調味料など）に整理。
    *   カメラ撮影またはフォトライブラリから**食材の写真を追加**可能。サムネイル表示で一目で把握できます。
    *   `＋` `－` ボタンのワンタップで直感的に数量の増減が可能。
    *   消費期限に応じてカードの枠線が変化（赤：期限切れ、黄：期限間近、緑：安全）。
3.  **🍽 晩ごはんの記録 ＆ スマート消費連携**
    *   作った料理の名前、日付、星評価、写真、レシピメモを登録可能。
    *   **iOS純正メモアプリ風のMarkdown（マークダウン）表示対応！** 箇ろ書き、太字、手順番号などが見やすく表示され、レシピ記録に最適。
    *   **【自動減算システム】**: 晩ごはんを記録する際、使った食材をチェックボックスで選ぶと、**冷蔵庫の在庫数が自動でマイナス（または使い切って削除）**されます！

## 🛠 ご自身のiPhoneやシミュレータで動かす手順

このフォルダ内には、アプリの全ソースコードが美しく配置されています。Xcodeを使ってご自身のデバイス（実機iPhoneまたはシミュレータ）で動かすのは非常に簡単です！

### 1. Xcodeでプロジェクトの新規作成
1.  Macで **Xcode** を起動します。
2.  **Create New Project** (またはメニューの `File` > `New` > `Project...`) を選択します。
3.  **iOS** タブの **App** を選択し、`Next` をクリックします。
4.  以下のように設定します：
    *   **Product Name**: `FridgeDinnerTracker`
    *   **Interface**: `SwiftUI`
    *   **Language**: `Swift`
5.  プロジェクトの作成先を、このフォルダと同じ階層（例: `/Users/kirinokazuya/Documents/ソースコード類/`）に指定して保存します。

### 2. ソースファイルのインポート
Xcodeプロジェクトが立ち上がったら、自動生成されたプレースホルダーファイルを上書き・追加します。

1.  Xcode左側のファイルツリーにある `ContentView.swift` を右クリックして **Delete** > **Move to Trash** で削除します。
2.  Finderでこの `FridgeDinnerTracker` フォルダを開き、以下のファイルとフォルダを、**Xcodeの左側ファイルツリーの `FridgeDinnerTracker` グループ内にドラッグ＆ドロップ**します：
    *   `FridgeDinnerTrackerApp.swift` (※すでにある同名ファイルを上書きするか、一度削除してから追加してください)
    *   `AppState.swift`
    *   `Models/` (フォルダごと)
    *   `Views/` (フォルダごと)
3.  追加する際のダイアログで、**"Copy items if needed" にチェックが入っていること**、および **"Create groups" が選択されていること** を確認し、`Finish` を押します。

### 3. 実機またはシミュレータでの実行！
1.  Xcode上部のツールバーで、実行対象デバイス（例: `iPhone 15 Pro` などのシミュレータ、またはケーブルやWi-Fiで接続したご自身の `iPhone`）を選択します。
2.  **⌘R** (またはツールバーの再生ボタン ▶) を押してビルドを開始します。
3.  数秒でアプリが起動し、すぐにサンプルデータ入りで美しいUIが動き出します！

## 🔒 セキュリティとデータ永続化について

このアプリは、外部のクラウドデータベースやサーバーを一切使用していません。
データ（食材リスト、履歴、写真など）はすべて**iPhone端末内のセキュアなDocuments領域に暗号化されたJSON形式で直接保存**されます。そのため：
*   オフラインでも完全に動作します。
*   個人情報や写真データが外部のインターネットに送信されることは一切ありません。プライバシーは100%守られます。

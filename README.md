## 💎 主な機能

1.  **📝 晩ごはん履歴の「タップ編集・削除」機能 [NEW]**
    *   履歴カードをタップして開く詳細画面（ディテールシート）に「編集」ボタンを搭載。
    *   タップすると記録モーダルが「編集モード」で起動し、**過去の記録内容（料理名、日付、星評価スライダー、写真、使った食材チェック、マークダウンレシピ）が全て完璧にフォーム内に復元**されます。
    *   修正を保存する際も、過去に減算された冷蔵庫の食材在庫数が不整合を起こさない安全ロジックを搭載。さらに、最下部の赤枠ボタンから一撃で記録を消去できる「削除」機能も完備。
2.  **ⓘ タイポグラフィが美しい「アプリ情報・データ解説」Info [NEW]**
    *   ダッシュボード右上に、クラシックなセリフ体（Georgia）で上品に丸囲みされた「*i*」ボタンを設置。
    *   タップすると、アプリの使い方とデータ保存（LocalStorage）の安全性・データ消失を防ぐコツ（シークレットモード回避など）を美しく解説する親身なモーダルが立ち上がります。
3.  **🎨 プレミアムな「ローカルPWAアプリアイコン」を標準搭載 [NEW]**
    *   冷蔵庫にオレンジのハートと新鮮な葉っぱをあしらった、極めてミニマルで美しい高解像度アイコン（`icon-512.png`）を新規設計し、ローカルに搭載。
    *   これにより、完全オフライン状態でもホーム画面に美しいアプリアイコンが完璧に表示され、ネイティブアプリ化が完成します。
4.  **📊 AI食事の構成要素分析（自動プロファイリング）**
    *   **手動入力・選択は一切不要！** 晩ごはん記録時に入力した「料理名」「レシピメモの記述」および「冷蔵庫から使用した食材」から、AIが裏側で食事の構成要素をバックグラウンドで完全自動抽出。
    *   夕食を7大要素（`鍋・煮込み`、`焼き物・炒め物`、`サラダ・和え物`、`汁物・スープ`、`揚げ物`、`麺類`、`丼物・ご飯物`）に分解し、星評価の高さで加重平均したバランスグラフ（プログレスバー）をダッシュボードに描画します。
5.  **💡 在庫 ✕ 好み連動の「今日の献立アイデア」**
    *   「現在の冷蔵庫で最も期限がやばい食材」と、AI分析による「よく作る/お気に入りの食事要素」をリアルタイムに自動マッチング。
    *   「期限間近の豚肉」と「サラダ好き傾向」を掛け合わせて『豚しゃぶ風の温野菜和えサラダ』を提案するなど、40パターン以上のバリエーションから千変万化するパーソナライズ献立推薦を行います。
6.  **⭐ 小数点第一位まで指定できる「星評価スライダー」**
    *   おいしさ評価を `1.0`〜`5.0` まで `0.1` 刻みで指定できるなめらかなiOS風レンジスライダーを搭載。
    *   重ね合わせCSS技術により、`4.2` などの小数評価に合わせて星の塗りがミリ単位で部分的に塗りつぶされる、極めて品格の高いグラフィック表示を完全再現。
7.  **🛒 個別スライド削除対応の「買い物メモ」**
    *   冷蔵庫リストで数量が「0」になった食材の下に表示される「メモに追加」ボタンから、ワンタップで自動的に買い物リストに転送。
    *   リストの行をタップすると右端から赤色の「削除」ボタンが滑らかにスライドインし、個別でスムーズに消去可能。

## 📱 スマートフォンでの導入・インストール方法

本アプリは **PWA (Progressive Web App)** に対応しているため、App StoreやGoogle Playを経由せず、ブラウザから簡単にスマホへ直接インストールできます。

### 🍎 iPhone / iPad の場合 (Safari)
1. **Safari** を開き、公開URL（またはローカルで起動したサーバーのIPアドレス）にアクセスします。
2. 画面下部にある **「共有」ボタン**（上矢印が飛び出した四角いアイコン）をタップします。
3. メニューをスクロールし、**「ホーム画面に追加」** をタップします。
4. 右上の **「追加」** をタップすると、ホーム画面に美しいアプリアイコン（冷蔵庫にオレンジハート）付きのアプリがインストールされます。
5. 以降はホーム画面からネイティブアプリとして、アドレスバーのない全画面表示かつ完全オフラインでもサクサクお使いいただけます。

### 🤖 Android の場合 (Google Chrome)
1. **Chrome** を開き、公開URLにアクセスします。
2. 画面下部または右上のメニュー（縦の3点リーダー）に表示される **「ホーム画面に追加」** または **「アプリをインストール」** をタップします。
3. 画面の指示に従ってインストールを完了します。
4. ホーム画面に冷蔵庫アイコンのアプリが追加され、ネイティブアプリのように全画面で起動します。

### 🧠 バックグラウンド解析のフロー
1.  **入力の要素分解**:
    晩ごはん記録時にチェックした「使用食材」や「レシピメモ」のテキストから、AIエンジン（`ai.js`）が調理法や具材ベース（大根・じゃがいも・出汁・醤油など）の要素を瞬時に検出。
2.  **星評価による重要度加重**:
    `log.rating`（スライダー値）の**二乗**を重みとして掛け合わせることで、適当に作った料理よりも、評価の高い「本当に大好きな食事要素」がグラフに強く反映されるように計算。
3.  **在庫スキャン ✕ バランス提案**:
    ダッシュボード描画時、現在の冷蔵庫在庫に「大根・豚肉・出汁」が揃っていることを自動検知すると、「🍲 鍋・煮込み」や「🥣 汁物・スープ」の食事要素が調理可能であると判断し、過去に不足している要素と組み合わせてスマートに推薦レシピを創り出します。

## 🚀 ローカル（パソコン）での起動方法
特別なビルドやサーバー構築は一切不要です！

1.  パソコンでこのフォルダを開きます。
2.  `index.html` ファイルをダブルクリック（またはGoogle ChromeやSafariなどのブラウザにドラッグ＆ドロップ）します。
3.  それだけでアプリが即座に起動し、LocalStorageの自動セーブ機能により、データを保持しながら即座に使いこなすことができます。

## 🔒 セキュリティとデータプライバシー

すべての写真、食材データ、レシピメモは、**お使いのスマートフォンやPCの内部セキュア領域（LocalStorage）に完全に閉じて自動保存**されます。
外部のクラウドサーバー等に一切データを送信しないため、他人に個人情報や写真を見られるリスクは100%ゼロであり、電波の届かないオフライン環境でも完全な機能性を維持して動作します。

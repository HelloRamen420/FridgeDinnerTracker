# 🍳 FridgeDinnerTracker - Webアプリ版 (PWA)

iPhoneやAndroidのブラウザからアクセスして、ホーム画面に追加するだけで本物のアプリとして動作する**モバイル特化型Webアプリ（PWA: Progressive Web App）**です。
開発環境やサーバー構築は不要で、お手持ちのスマートフォンで今すぐ動作します！

---

## ✨ 主な機能

1.  **📊 スマートダッシュボード**
    *   消費期限が近い食材（2日以内・期限切れ）をカード表示とバッジで自動警告。
    *   冷蔵庫の残り数、今週作った晩ごはんの総数をリアルタイム表示。
    *   **期限が近い食材を自動抽出し、おすすめ料理を日替わりで提案するスマート献立カード**を搭載！
2.  **🥩 冷蔵庫管理**
    *   食材をカテゴリ別（肉、魚、野菜、乳製品、調味料など）に整理。
    *   スマートフォンのカメラまたはフォトアルバムから**食材の写真を追加**してサムネイル表示。
    *   `＋` `－` ボタンのタップでスピーディに在庫数量を調整可能。
    *   期限ステータス（赤：期限切れ、黄：期限間近、緑：安全）が直感的なカラー枠線で変化。
3.  **🥘 晩ごはんの記録 ＆ スマート自動在庫減算**
    *   料理写真、調理日、星評価、レシピメモをまとめて記録。
    *   **箇条書き・太字・手順リストが綺麗に表示される「iOSメモ風Markdownレンダラー」を搭載！** ライブプレビュー機能付きでレシピが書きやすい。
    *   **【自動在庫減算システム】**: 晩ごはんを記録する際、使った冷蔵庫の食材を選択すると、**在庫数から自動でマイナス（または使い切った場合は冷蔵庫から自動削除）**されます！

---

## 🚀 ローカル（パソコン）での起動方法
特別なビルドやインストール作業は一切不要です！

1.  パソコンでこのフォルダを開きます。
2.  `index.html` ファイルをダブルクリック（またはGoogle ChromeやSafariなどのブラウザにドラッグ＆ドロップ）します。
3.  それだけでアプリが起動し、すぐにサンプルデータと共にすべての機能が動作します。

## 🌐 自分のiPhoneやご家族・ご友人に無料公開する手順（GitHub Pages）

すでに作成したGitHubリポジトリ（`HelloRamen420/FridgeDinnerTracker`）の「GitHub Pages」機能を使うことで、**完全無料・ワンクリック**で全世界に公開し、スマホでアクセスしてホーム画面に「アプリ」として追加できます。

### ステップ1: 変更をGitHubにプッシュする
このフォルダ（Web版が入っているフォルダ）で、ターミナルから以下のコマンドを実行してGitHubにアップロードします。
```bash
git add .
git commit -m "feat: migrate to premium web app PWA version"
git push origin main
```

### ステップ2: GitHub上でWeb公開の設定をする
1.  ブラウザでご自身のGitHubリポジトリ（ `https://github.com/HelloRamen420/FridgeDinnerTracker` ）を開きます。
2.  上部メニューの ⚙️ **Settings** をクリックします。
3.  左側サイドバーの **Pages**（または **Code and automation** > **Pages**）を選択します。
4.  **Build and deployment** の設定で、以下のように選択します：
    *   **Source**: `Deploy from a branch`
    *   **Branch**: `main` / `/ (root)`
5.  右側の **Save** ボタンをクリックします。

### ステップ3: スマホでアプリを遊ぶ！
1.  保存後、1〜2分待つと、画面上部に公開URL（例: `https://helloramen420.github.io/FridgeDinnerTracker/` ）が表示されます。
2.  このURLをご自身のiPhoneの **Safari**（Androidの場合はChrome）で開きます。
3.  Safariの下部メニューから **「共有」ボタン**（上矢印のアイコン）をタップし、**「ホーム画面に追加」** を選択します。
4.  ホーム画面にアイコンが追加され、次からはそれをタップするだけで、アドレスバーのない**本物のフルスクリーンアプリとしていつでも快適に利用できます！**

## 🔒 セキュリティとデータ永続化について

このアプリは、外部のサーバーやデータベースに一切データを送信しません。
食材の在庫リストや追加した写真などのすべてのデータは、**お使いのスマートフォンのブラウザ内セキュア領域（LocalStorage）に自動保存**されます。
*   他人に写真やデータを見られる心配がなく、プライバシーは完全に安全です。
*   飛行機内などのオフライン状態でも、全く問題なく動作します。
